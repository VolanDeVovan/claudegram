# UX Flows — примеры взаимодействия

## Настройка через чат

> "Добавь проект /home/me/frontend, назови react-app"

Claude вызывает `config_get("projects")` → добавляет в массив → `config_set("projects", [...])` → "Готово, проект react-app добавлен. Сейчас переключение между проектами не настроено. Ты можешь выбрать способ: команды, inline-кнопки, форум-треды — или предложить свой. Скажи как удобнее, я настрою."

> "Покажи все настройки"

Claude вызывает `config_get` → возвращает читаемый список.

> "Поставь таймаут 10 минут и модель sonnet"

Claude вызывает `config_set` дважды → "Таймаут: 10 мин, модель: claude-sonnet-4. Применено."

## Управление плагинами

> "Сделай чтобы бот использовал треды — каждый проект в отдельном треде"

Claude:
1. Читает `src/templates/thread-routing.ts` — видит готовый шаблон
2. Копирует в `plugins/thread-routing.ts`
3. Адаптирует (читает config, маппит проекты на thread IDs)
4. Вызывает `reload_plugins` → ядро создаёт поколение, перезагружает
5. "Готово. Каждый проект теперь в отдельном треде."

> "Выключи rate limiting"

Claude:
1. Удаляет `plugins/rate-limit.ts`
2. Вызывает `reload_plugins`
3. "Rate limiting выключен. Если что, могу откатить."

> "Какие плагины активны?"

Claude вызывает `plugin_list` → возвращает список активных и доступных шаблонов.

> "Включи команду /status"

Claude копирует `src/templates/status-command.ts` → `plugins/status-command.ts` → `reload_plugins` → "Готово, /status работает."

> "Сделай команду /deploy которая запускает make deploy"

Claude пишет новый плагин с нуля, ориентируясь на паттерны из templates → `reload_plugins` → "/deploy готов."

## Откат

> "Верни как было вчера"

Claude:
1. Вызывает `generation_list` → видит историю
2. Вызывает `generation_rollback(gen-005)` → ядро восстанавливает plugins/
3. Reload
4. "Откатил до поколения 5 (вчера, 14:30). Активные плагины: auth, text-handler, ack-reaction."

## Управление доступом

> "Добавь юзера 123456"

Claude (через тул user_add плагина auth):
1. `config_set("plugins.auth.allowedUsers", [...current, "123456"])`
2. "Юзер 123456 добавлен. Теперь он может писать боту."

> "Кто имеет доступ?"

Claude: `config_get("plugins.auth.allowedUsers")` + `config_get("owner")` → "Owner: 412587349. Дополнительные юзеры: 123456."
